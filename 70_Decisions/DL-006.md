# DL-006: ObservatoryType follows repo docs values

## Status
Accepted

## Context
Під час ISSUE-03 зафіксовано фактичні робочі значення типу ObservatoryType у репозиторії.

## Decision
- Поточна implementation truth для `ObservatoryType`:
  - `individual`
  - `studio`
  - `product`
- Старі альтернативні enum-варіанти не є активною implementation truth, доки не будуть окремо формально промотовані.

## Consequences
- Валідація, seed-дані, та документація мають орієнтуватися на ці три значення.
- Будь-які зміни переліку мають проходити через окреме рішення та оновлення vault.
