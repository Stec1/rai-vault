# DL-001: Documentation as Operating Source of Truth

## Status
Accepted

## Context
Execution relies on multi-agent handoffs and continuity across sessions.

## Decision
Use the vault documentation as the primary operating memory system.

## Consequences
- Requires disciplined update cadence.
- Improves resilience when chat context is lost.
