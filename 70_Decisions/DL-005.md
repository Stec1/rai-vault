# DL-005: Prisma 7 adopted in API layer

## Status
Accepted

## Context
У межах ISSUE-03 фактична реалізація API-шару виконана на Prisma 7.x.

## Decision
- Prisma 7.x є поточною implementation truth для API layer.
- `prisma.config.ts` є частиною актуального runtime/setup-контуру Prisma.
- Майбутні технічні документи та execution prompts мають бути узгоджені з реальністю Prisma 7.x.

## Consequences
- Старі припущення про конфігурацію Prisma мають вважатися застарілими.
- Опис архітектури та інструкції виконання потребують підтримки в актуальному стані.
