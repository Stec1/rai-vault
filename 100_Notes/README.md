# 100_Notes — Exploratory Zone

## What this is
A space for ideas, hypotheses, and founder strategic thinking.

## Rules
1. 100_Notes is NOT a source of truth
2. Ideas here are NOT approved decisions
3. When an idea becomes a decision — it must be promoted to `70_Decisions/` via the Decision promotion process
4. Free format allowed, but each note must have a date and a status: idea / exploring / ready-for-review / promoted / archived

## Idea vs Decision
- Idea: "What if we add a marketplace?" → 100_Notes
- Decision: "Marketplace will be in Phase 3. Rationale: ..." → 70_Decisions/DL-XXX.md

## Structure
- `Ideas_Backlog.md` — general idea list
- `Scaling_Hypotheses.md` — growth and scaling hypotheses
- `Business_Model_Notes.md` — monetization thinking
- [Any new .md files as needed]

## How to avoid chaos
- Every note has a date tag
- Archive or delete stale notes monthly
- Do not create a file per micro-idea — group into thematic files
- If a file grows past 50 items — split into sub-files
